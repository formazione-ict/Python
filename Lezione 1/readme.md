# Corso Python: Fondamenti e Strutture Dati (Liste)

Benvenuti nel repository del corso. Questo materiale copre l'utilizzo delle **Liste** in Python, una delle strutture dati più fondamentali e versatili del linguaggio.

## Contenuti

1. **Basi (`01_basi_liste.py`)**: Come creare liste, tipi di dati misti, indicizzazione (positiva e negativa).
2. **Metodi (`02_metodi_liste.py`)**: Aggiungere (`append`, `insert`), rimuovere (`remove`, `pop`) e ordinare (`sort`) gli elementi.
3. **Slicing e Loop (`03_slicing_loop.py`)**: Estrarre porzioni di lista e utilizzare cicli `for` e `enumerate`.
4. **List Comprehension (`04_list_comprehension.py`)**: Scrivere codice più conciso e "Pythonic".
5. **Progetto Pratico (`progetto_voti.py`)**: Un piccolo programma interattivo per gestire voti scolastici e calcolare statistiche.

## Requisiti

- **Python 3.8+** installato sulla macchina.
- Un IDE a scelta tra **PyCharm**, **VS Code** o **Thonny**.

## Come eseguire il codice

Apri il terminale nella cartella del progetto ed esegui:

```bash
python 01_basi_liste.py
