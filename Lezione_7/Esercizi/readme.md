# Utilizzo del template
Questo template è potente per la didattica
## ✔ Struttura modulare
Gli studenti vedono chiaramente:
- dati
- figura
- plot
- stile
- export
- show

Questo riduce confusione e aumenta la leggibilità.
## ✔ Anticipa errori tipici
- cartella outputs/ sempre creata
- export sempre prima di plt.show()
- OO API obbligatoria
- legenda funzionante grazie ai label=
## ✔ Riutilizzabile per tutti gli esercizi
Basta cambiare:
- i dati
- i plot
- il nome del file
Il resto rimane identico.
## ✔ Subplot multipli con layout complesso
- griglia 2×2
- subplot largo in basso
- gestione degli spazi con constrained_layout=True
## ✔ Stili e temi professionali
- uso di plt.style.use()
- colori, linewidth, linestyle
## ✔ Tecniche avanzate
- annotazioni con frecce
- ticks personalizzati
- limiti assi
- scatter + line plot combinati
- rumore per simulare dati reali
## ✔ Esportazione pulita
- bbox_inches='tight'
- dpi=200
- cartella outputs/ garantita
## ✔ Struttura didattica perfetta
Ogni blocco è separato e commentato, così è possibile capire cosa fanno e perché.
